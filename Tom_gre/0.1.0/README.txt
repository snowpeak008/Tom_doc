Tom 0.1.0 安装版

打开安装包：

Tom_Setup_0.1.0.exe

安装结果

- 安装位置：%LOCALAPPDATA%\Tom
- 桌面快捷方式：Tom
- 开始菜单：Tom\Tom

说明

Tom 是本地AI文档工作台，用于编写、整理和保存玩法文档、需求文档、说明文档等内容。

选中文字后，在选区内右键可直接执行扩写、美化、总结、需求。
AI 页签使用自由指令主按钮，不再显示扩写、美化、总结、需求、检查快捷按钮。
AI 运行中会显示加载状态，自由指令按钮暂时不可重复点击，输入框仍可编辑。
AI 历史支持查看修改前、修改后和 Diff，也可以一键清理历史。

Tom 不内置 Codex 或 Claude。如果电脑已经安装并登录 Codex CLI 或 Claude CLI，Tom 会自动检测并使用；如果没有安装，仍可离线编辑、保存、导入和导出文档。

卸载

安装后可在安装目录中运行：

%LOCALAPPDATA%\Tom\Uninstall_Tom.cmd

卸载脚本默认只删除程序和快捷方式，会保留 tom-docs 用户数据目录。
