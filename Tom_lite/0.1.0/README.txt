Tom - 本地AI文档工作台

Tom 是一个在 Windows 上运行的本地文档工具，用来编写、整理和保存玩法文档、需求文档、说明文档等内容。

如何打开

双击：

Tom.exe

首次打开后，Tom 会在当前目录旁边创建 tom-docs 文件夹，用来保存你的文档、快照、导出文件和 AI 记录。

主要功能

- 编写和保存本地文档
- 新建、打开、另存多个文档
- 标题、正文、加粗、斜体、下划线、对齐、编号和项目符号
- 查找和替换
- 导入 TXT、Markdown、HTML、RTF、DOCX
- 导出 TXT、Markdown、HTML、RTF、DOCX
- 创建和恢复快照
- 通过大纲快速跳转章节
- 使用 Codex 或 Claude 辅助扩写、美化、总结、整理需求和检查文档
- 选中文字后，在选区内右键可直接执行扩写、美化、总结、需求
- AI 历史支持查看修改前、修改后和 Diff 差异，也可以一键清理历史

AI 环境说明

Tom 不内置 Codex 或 Claude。

如果你的电脑已经安装并登录 Codex CLI 或 Claude CLI，Tom 会自动检测并使用。
如果没有安装，Tom 仍然可以离线编辑、保存、导入和导出文档，只是 AI 功能不可用。

目录说明

- Tom.exe：启动程序
- README.txt：当前说明文件
- tom-docs：你的文档数据目录
- debug-symbols：开发调试文件，普通使用可以忽略

注意

不要随意删除 tom-docs 文件夹。你的文档、快照和导出记录都保存在里面。
